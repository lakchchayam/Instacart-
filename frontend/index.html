The Real-World Problem

Grocery retailers rarely provide real-time shelf inventory APIs. This creates a "black box" where Instacart doesn't know if an item is truly in stock until a shopper reaches the aisle.

Methodology: Availability Scorer

We solve this by treating Shopper Found/Not Found logs as a real-time data stream. Using a decay-weighted average, we calculate the Availability Probability Score. If multiple shoppers fail to find an item within a short window, the model flags it as Out-Of-Stock (OOS) for future customers.

Methodology: Bayesian Substitution

When an item is OOS, our ranking engine suggests replacements by calculating the joint probability of Category Fit and Substitute Availability. This prevents "bad trades" that lead to customer refunds.
